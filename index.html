Uxkode Product Addons for WooCommerce

Uxkode Product Addons is a lightweight and developer-friendly plugin that helps you enhance your WooCommerce store with two powerful features — customizable Product Add-Ons and flexible Custom Buttons.

Getting Started

Follow these steps to install and activate the plugin:

  1. Download and install the plugin from WordPress.org or upload the ZIP file via Plugins → Add New.
  2. Activate the plugin from your WordPress dashboard.
  3. Go to Uxkode Addons Dashboard to manage Product Add-Ons, Custom Buttons, and all plugin settings from one dashboard.

Key Features

  • Manage Product Add-Ons from a dedicated Product Add-Ons admin dashboard.
  • Create Add-Ons with multiple field types (text, number, textarea).
  • Assign available Add-Ons to products via the Product Edit → Product Add-Ons tab.
  • Enable or disable Add-Ons per product. When enabled and assigned, they will display on the product page.
  • Customize the appearance of Custom Buttons globally from the Custom Buttons admin dashboard.
  • Enable or disable the Custom Buttons feature per product and choose between single or dual call-to-action buttons.
  • Set button label, URL, and target (same tab or new tab) directly from the Product Edit → Custom Buttons tab.
  • Built with performance in mind — clean, lightweight, and developer-friendly.

Plugin Mimumum Requirements

  • WordPress 5.8 or later
  • WooCommerce 6.0 or later
  • PHP 7.4 or later

Configure Product Add-Ons

To create and manage product Add-Ons:

  1. Go to Uxkode Addons Dashboard → Product Add-Ons.
  2. Fill Up the Add-Ons form with the necessary details and click 'Create Add-On'
  3. Assign custom available Add-Ons to specific WooCommerce products from (Product Edit screen → Product Add-Ons tab).
  4. Publish New Product or Update changes — Add-Ons will automatically appear on the single product page.

Custom Buttons

To enable and manage custom buttons for a product:

  1. Go to Uxkode Addons Dashboard → Custom Buttons.
  2. Add custom styles and hover effects from the Custom Buttons admin dashboard.
  3. Edit the desired product and go to Custom Buttons tab to enable/disable the feature and choose between Single Button or Dual Buttons. Add button details as needed.
  4. Publish New Product or Update changes — your custom buttons will now appear on the single product page.

FAQ

Will this make my site slower?

No. Uxkode Product Addons is built for performance and only loads assets where they are needed. No unnecessary CSS or JS files are added to your site.

Do Product Add-Ons work with variable products?

Yes! Product Add-Ons work perfectly with both simple and variable products, so you can easily collect customer inputs and offer extra customization options.

Can I enable Add-Ons for specific products only?

Absolutely. You can enable Add-Ons on a per-product basis and assign specific ones from the Product Edit → Product Add-Ons tab.

Can I customize the look of the buttons?

Yes! You can style both Single and Dual Buttons directly from the Uxkode Addons Dashboard → Custom Buttons. Advanced users can also override styles with custom CSS for complete control.

Can I add two buttons to a single product?

Yes. You can display dual buttons on a product and configure their text, URL, and open behavior (same tab or new tab) from the Product Data tab.

Troubleshooting

Tip: If Add-Ons or Custom Buttons do not appear, clear your site cache and confirm that the feature is enabled for that product.
  • Enable WP_DEBUG in wp-config.php to check for errors.
  • Temporarily deactivate other plugins to rule out conflicts.
  • Ensure WooCommerce is active and updated to the latest version.
